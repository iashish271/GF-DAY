A LITTLE UNIVERSE MADE JUST FOR YOU
====================================

A premium cinematic interactive love story website.
Built with pure HTML, CSS, and Vanilla JavaScript.
No frameworks. No libraries.

FOLDER STRUCTURE:
-----------------
universe-love/
├── index.html          # Main HTML structure
├── style.css           # All styles & animations
├── script.js           # All JavaScript logic
├── song.mp3            # Background music (add your own!)
└── images/
    ├── photo1.jpg      # Memory photos
    ├── photo2.jpg
    └── photo3.jpg

HOW TO USE:
-----------
1. Place your photos in the /images folder
2. Add your song.mp3 to the root folder
3. Open index.html in any modern browser
4. The default password is "love"

CUSTOMIZATION:
-------------
- Password: Edit CONFIG.password in script.js
- Letter text: Edit CONFIG.letterText in script.js
- Wrong messages: Edit CONFIG.wrongMessages in script.js
- Photo captions: Edit CONFIG.photoCaptions in script.js
- Colors: Edit CSS variables in style.css :root

FEATURES:
---------
INTRO:
- Black screen with 300 twinkling stars
- One star glows brighter after 3 seconds
- Typewriter text: "Among billions of stars..."
- Click the star for magical explosion & nebula

PASSWORD PAGE:
- Floating rotating planet with golden ring
- Magical lock on planet
- Wrong password: planet shake, meteor fall, cute GIF popup
- Correct password: planet cracks, golden light, heart burst

SPACE TRAVEL:
- Camera flies through galaxy, nebula, Saturn, moon
- Shooting stars, floating hearts, aurora

WELCOME PAGE:
- Dream planet with pink sky
- Cotton candy clouds, floating islands

MEMORY SECTION:
- Floating crystal memories
- Click crystal to open photo modal

SECRET LETTER:
- 3D envelope that opens
- Letter slides out with typewriter animation
- Handwriting font

SURPRISE PAGE:
- Gift box with ribbon

ENDING:
- Calm galaxy with moon

EXTRA:
